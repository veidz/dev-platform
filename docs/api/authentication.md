# API - Autenticação

Descrição dos fluxos de autenticação e autorização.

## Estratégia

- JWT para autenticação
- RBAC para autorização

## Endpoints

- POST /auth/login
- POST /auth/refresh
- POST /auth/logout

## Tokens

- Access token (15 min)
- Refresh token (7 dias)

> Detalhar payloads e exemplos quando os serviços forem implementados.
